# Dreftinput Website

Sitio estatico de presentacion para Dreftinput.

## Archivos

- `index.html` contiene la estructura principal.
- `styles.css` define el diseno glass, responsive y animaciones.
- `script.js` controla la demo, el cambio de tema, el cursor glow y los efectos de scroll.
- `assets/` contiene los visuales SVG usados por la pagina.

Abre `index.html` directamente en el navegador para revisar la pagina.
