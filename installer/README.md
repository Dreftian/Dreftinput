# Instalador de Dreftinput

El instalador comercial recomendado es el launcher empaquetado:

- `release/Dreftinput-Launcher-1.0.0.zip`

Ese archivo contiene `Dreftinput Launcher.exe` y el payload local de la aplicacion. El launcher presenta Dreftinput, instala la app en archivos locales y crea el acceso directo de Windows.

## Publicacion

El paquete pesa mas de 280 MB, por eso no debe subirse como archivo normal del repositorio. Publicalo como asset de GitHub Releases con este nombre exacto:

`Dreftinput-Launcher-1.0.0.zip`

La pagina web usa esta URL de descarga:

`https://github.com/Dreftian/Dreftinput/releases/latest/download/Dreftinput-Launcher-1.0.0.zip`
